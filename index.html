Chainer orchestrates Claude Code plugins into automated workflows. Chain planning, implementation, testing, and deployment into single commands.

/chainer:run worktree-plan-implement \ --feature_name="oauth" \ --prompt="Build OAuth2"

The Concept

Turn plugin sequences
into single commands.

Instead of manually running multiple plugins in sequence, define a chain once and execute your entire workflow with one command.

Chainer orchestrates any Claude Code plugins. Create worktrees, plan features, implement code, run tests, and deploy—all automated through YAML configuration.

Capabilities

Designed for multi-step workflows that benefit from automation.

  • Chain any Claude Code plugins in sequence
  • Config-driven workflows with YAML
  • Variable substitution between steps
  • Parallel execution with tmux support
  • Community chain library

Installation

One line setup.

Requires Claude Code. Works best with Worktree Manager, feature-dev, and ralph-wiggum.

# Clone the repo git clone https://github.com/danielraffel/Chainer ~/.claude/plugins/chainer # That's it - Chainer is automatically detected cd /your/project claude

Built-in Chains

Ready to use.

worktree-plan-implement
/chainer:run worktree-plan-implement \ --feature_name="payments" \ --prompt="Stripe integration"
Full workflow: creates worktree, plans, and implements.
plan-and-implement
/chainer:run plan-and-implement \ --prompt="Add user auth" \ --feature_name="auth"
Plans with feature-dev, implements with ralph-wiggum.
plan-only
/chainer:run plan-only \ --prompt="Design API" \ --feature_name="api"
Just planning, no implementation.
implement-only
/chainer:run implement-only \ --spec_file="audit/api.md"
Implements from existing spec file.

Custom Chains

Build your own workflows.

Define chains in ~/.claude/chainer.local.md using simple YAML. See the community chains library for examples.

# ~/.claude/chainer.local.md --- chains: my-workflow: enabled: true description: "Custom development workflow" inputs: task: { required: true } steps: - name: plan type: skill skill: feature-dev:feature-dev args: "{{task}}" - name: test type: script script: npm test - name: build type: script script: npm run build ---
